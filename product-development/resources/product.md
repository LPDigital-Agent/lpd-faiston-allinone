# Faiston NEXO - Product Overview

## Company Background

**Faiston** é uma empresa de tecnologia brasileira especializada em soluções de Cloud, AI e infraestrutura de TI. Com uma equipe técnica altamente qualificada, a Faiston atende clientes enterprise em transformação digital.

## Product: Faiston NEXO

### What is NEXO?

NEXO é a plataforma AI-First All-in-One Intranet da Faiston. Mais do que uma intranet tradicional, NEXO é um hub inteligente comandado por um assistente de IA que orquestra todas as interações, integrações e automações para os funcionários.

### Product Philosophy

**AI-First**: Toda funcionalidade é projetada com IA no centro. O assistente NEXO não é um add-on, é o comandante da plataforma.

**All-in-One**: Um único ponto de acesso para todas as informações e ferramentas necessárias no dia-a-dia.

**Minimum Lovable Product (MLP)**: Não apenas funcional, mas delightful. A interface é projetada para ser amada, não apenas usada.

### Core Values

1. **Velocidade**: Operação instantânea, sem latência perceptível
2. **Inteligência**: IA que antecipa necessidades e automatiza tarefas
3. **Unificação**: Um único ponto de acesso para tudo
4. **Elegância**: Design state-of-the-art que representa a Faiston

## Target Audience

- **Primary**: Todos os funcionários da Faiston
- **Secondary**: Parceiros e stakeholders externos (futuro)

## Competitive Landscape

| Solution | Type | Limitation |
|----------|------|------------|
| SharePoint | Traditional Intranet | Não é AI-First, UX datada |
| Notion | Workspace | Não tem integração enterprise nativa |
| Slack/Teams | Communication | Não é portal unificado |
| Custom Intranet | Development | Geralmente sem AI |

**NEXO Differentiator**: Único portal que combina IA generativa (NEXO Assistant) + integrações enterprise (Office 365) + curadoria inteligente de conteúdo + UX moderna.

## Technology Stack Summary

- **Frontend**: Next.js 15, TypeScript, Tailwind CSS
- **Backend**: Python 3.12, Google ADK, AWS Lambda
- **AI Runtime**: AWS Bedrock AgentCore
- **Infrastructure**: AWS (Terraform), GitHub Actions
